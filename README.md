# A* Maze Solver

An interactive visualization of the **A\*** (A-star) pathfinding algorithm on a grid-based maze.  
The program finds and displays the shortest path from a start cell to a goal cell.

---

## ✨ Features
- Visualizes the step-by-step execution of the A* algorithm  
- Interactive grid where you can set **start** and **goal** positions with mouse clicks  
- Cells change color to indicate **open**, **closed**, and **path** states  

---

## 🛠️ Technologies
- **C++17**  
- **SFML** (Simple and Fast Multimedia Library) for graphics  
- Standard Template Library (STL)  

---

## 🚀 Getting Started

### Prerequisites
- A C++ compiler with C++17 support (g++, clang, or MSVC)  
- [SFML](https://www.sfml-dev.org/) installed (graphics, window, and system modules)  
- Git
